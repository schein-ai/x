# ホームページ コピー — 日本語版（モデル版）

> `index-jp.html` の確定コピー一式。今後の他ページ／他言語の **モデル** として参照する。

## トーンモデル（重要）

このコピーで確立したスタイルを他ページにも適用する:

| 要素 | スタイル |
|---|---|
| **Hero ヘッドライン** | 短く・骨太・印象的（「AI で、想像を、実装する。」） |
| **セクションタイトル** | 直接的（「AI で実装する、4 つの事業。」「xR&D が選ばれる、4 つの理由。」など）|
| **説明文** | 1〜2 文・40〜60 文字程度のミディアム長。主語と動詞を明示 |
| **クオート系** | 例外的に長文も OK（About の Tomoki クオートのように、哲学を語る場面） |
| **読点 `、` の使い方** | リズムを生む装置（「AI で、想像を、実装する。」のように） |

### NG パターン（過去の失敗から）
- ❌ 短文断片の連結（「実践する。だから、届けられる。」 — 抽象すぎ）
- ❌ 詩的すぎ・抽象すぎ（「私たちは、作る側にいる。」を sec-title にする等）
- ❌ 説明文を 80 文字以上に伸ばす（読みにくくなる）

## モデルとしての要点（他ページ・他言語へ展開する際のチェックリスト）

- [ ] **Hero** は短く骨太、印象的な動詞・名詞を 1 つだけ強調
- [ ] **セクションタイトル**は直接的で「何のセクションか」が即わかる
- [ ] **説明文**は 40〜60 文字、主語と動詞を明示
- [ ] **読点**でリズムを作る（特にタイトル）
- [ ] **クオート**は例外的に長文 OK（哲学・物語の場）
- [ ] 「**研究**」は使わない（社名 R&D は OK）
- [ ] 「**弊社**」は使わない（一人称: 私たち／xR&D）
- [ ] **AI ネイティブ**を中心キーワードに据える
